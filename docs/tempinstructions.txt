Hybrid “snabb copytrade”.

1. Prediktiv filtrering av nya pooler några sekunder före Cupsyys köp.
2. Bekräftelse så snart Cupsyys första signatur syns.

INFRASTRUKTUR

* Läsflöde:
  Yellowstone gRPC Geyser‑plugin på Chainstack. Strömmen kommer direkt från validatorminnet och levereras i millisekunder enligt Chainstack‑beskrivningen ([Chainstack][1]).
  (Latency i praktiken: cirka 20‑60 ms efter slot‑commit; fortfarande tiotals millisekunder snabbare än vanlig WebSocket).
* Sändväg:
  Jito Block Engine, metod sendBundle, header x‑jito‑auth med egen UUID.
* Serverplacering: Amsterdam (EU C) eller Ashburn (US E) med ping under 20 ms.

KOSTNAD
Yellowstone gRPC tillägg på Chainstack Global Node kostar för närvarande från 149 USD per månad ([Chainstack][1]).

1. Pool‑init via Yellowstone Geyser → beräkna features och ML‑score.
2. Bygg och signera flera swap‑transaktioner med olika priority‑fee‑nivåer och håll dem i minnet.
3. När Cupsyys signatur bekräftas → skicka snabbaste bundle via Block Engine med hög tip.
4. Sälj när priset når exit‑regel (t.ex. pris har ökat 15%) eller när Cupsyy börjar skala ut.

* Stoppa botten och larma om rullande precision på de senaste 50 affärerna sjunker under 85 %.
